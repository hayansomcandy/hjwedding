# Wedding Invitation Project

## 1. 시작하기 전에 (Prerequisites)
이 프로젝트는 **React**로 만들어졌으며, 실행하려면 컴퓨터에 **Node.js**가 설치되어 있어야 합니다.
현재 시스템에 Node.js가 감지되지 않았습니다.

1. [Node.js 공식 홈페이지](https://nodejs.org/)에서 **LTS 버전**을 다운로드하여 설치해주세요.
2. 설치 후 터미널(VS Code 등)을 재시작해주세요.

## 2. 설치 및 실행 (Installation)
폴더 위치에서 터미널을 열고 다음 명령어를 입력하세요.

```bash
# 1. 라이브러리 설치
npm install

# 2. 개발 서버 실행
npm run dev
```

터미널에 나오는 주소(예: `http://localhost:5173`)를 클릭하면 브라우저에서 확인할 수 있습니다.

## 3. 기능 안내
### 💌 카카오톡 공유
- `index.html` 파일의 `<meta property="og:..."` 부분을 수정하여 썸네일과 제목을 변경하세요.
- 메인 화면의 [공유하기] 버튼을 누르면 모바일에서는 기본 공유 시트(카카오톡 포함)가 열리고, PC에서는 링크가 복사됩니다.

### 📋 RSVP (참석 여부 전달) & 관리자 모드
- 하객들이 "참석 여부 전달" 폼을 작성하면 브라우저 내부 저장소(LocalStorage)에 저장됩니다.
- **관리자 페이지 접속 방법**:
  1. 화면 맨 아래 `Footer`에 있는 작고 희미한 `Admin` 글씨를 클릭하거나,
  2. 주소 뒤에 `?admin=true`를 붙여 접속하세요. (예: `http://localhost:5173/?admin=true`)
- 관리자 페이지에서 **[Excel 다운로드]** 버튼을 누르면 명단을 엑셀 파일로 받을 수 있습니다.

## 4. 지도 설정
- `src/components/Location.jsx` 파일을 열어보시면 카카오맵 연동을 위한 주석이 있습니다.
- 실제 지도를 띄우려면 카카오 개발자 사이트에서 API 키를 발급받아야 합니다.
